# Kursova_NVBD_Variant_10

1. Опис проєкту
Даний репозиторій містить матеріали курсової роботи з дисципліни
«Нереляційні та великі бази даних».  
Метою роботи є проєктування та реалізація інформаційно-аналітичної системи
з використанням реляційної бази даних, ETL-процесів, OLAP-куба та аналітичних звітів.

У межах курсової роботи реалізовано:
- структуру бази даних;
- скрипти створення таблиць, обмежень, процедур та тригерів;
- процеси завантаження та трансформації даних (SSIS);
- OLAP-куб для аналітичного аналізу (SSAS);
- аналітичні звіти (SSRS).

2. Автор

3. Використані технології
- Microsoft SQL Server  
- SQL Server Integration Services (SSIS)  
- SQL Server Analysis Services (SSAS)  
- SQL Server Reporting Services (SSRS)  
- Visual Studio  

4. Інструкція з розгортання проєкту

1. Відкрити Microsoft SQL Server Management Studio.
2. Запустити скрипт `CREATE_DATABASE.sql`та `CREATE_DATABASE.sql` для створення бази даних та таблиць.
3. Згенерувати та завантажити тестові дані за допомогою скриптів з папки `DataGeneration`.
4. Відкрити проєкт SSIS у Visual Studio та виконати ETL-пакети.
5. Розгорнути OLAP-куб за допомогою проєкту SSAS.
6. Переглянути аналітичні звіти у середовищі SSRS.

5. Результати виконання роботи
- Реалізовано базу даних відповідно до предметної області.
- Побудовано процеси ETL для завантаження даних.
- Створено OLAP-куб для багатовимірного аналізу.
- Реалізовано аналітичні звіти для візуалізації даних.

6. Документація
Фінальний звіт курсової роботи з детальним описом реалізації та скріншотами
знаходиться у папці `Documentation`.

7. Додаткові матеріали
Скріншоти структури бази даних, ETL-процесів, OLAP-куба та звітів
наведені у звіті курсової роботи.
